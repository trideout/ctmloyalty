# CTM Loyalty application

### Installation
Clone Git repo into a directory.
Copy the .env.example to .env and configure database settings
Run `artisan key:generate`
Run `artisan serve` or host and run on local nginx instance

### Functionality
CRUD endpoints available at /api/subscribers for GET/POST/PATCH/DEL

Example usage available in postman sample file in `./Loyalty.postman_collection.json` 

### Database Scema
```
+-------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

| subscribers | CREATE TABLE `subscribers` (
  `id` bigint unsigned NOT NULL AUTO_INCREMENT,
  `email` varchar(255) COLLATE utf8mb4_unicode_ci NOT NULL,
  `opt_in` tinyint(1) NOT NULL DEFAULT '0',

```
